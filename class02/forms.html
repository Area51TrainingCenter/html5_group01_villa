Estas medidas no son obligatorias, y son alterables en CSS.

Algunos atributos:

  • placeholder="Nombre" → Esto coloca una instrucción/tip en el campo.
  • value="Ronald" → Cuando el usuario llena un campo de texto, por ejemplo, la información ingresada es el "value". Se puede pre-escribir el value con este atributo, pero el usuario sigue siendo capaz de modificarlo.
  • disabled → Esto desactiva el campo.
  • required → Esto hace que el campo sea requerido (La validación se hace vía Javascript).
  • hidden → Esto oculta el campo al usuario, pero puede llevar información.

Tip adicional:

Escribir el input field DENTRO de la etiqueta "label" también es válido, y aporta en una mejor experiencia de usuario. Al dar click sobre el texto del label, el input se vuelve activo.
